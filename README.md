# OpenCV mouse control


## Introduction

##### Control your mouse via hand gestures

## Requirements
* python3.7

* opencv-python, numpy, mediapipe

* screeninfo

## Usage

#### Firstly, install all required libs

```shell
$ python3 -m pip install -r requirements.txt
```
#### To run the app:

```
$ python3 main.py
```
